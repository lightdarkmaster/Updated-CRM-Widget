# Zoho Embedded App

This repository contains the source code for the Zoho Embedded App/Custom Widget, a custom widget built on top of the Zoho CRM platform, tailored to meet the specific business requirements of a client. The app is designed to provide a seamless and integrated experience for users, by combining the power of Zoho CRM with the flexibility of a custom widget.

## Prerequisites

*   Zoho CRM Account
*   Zoho CRM Developer Account
*   Zoho CRM API Key
*   Zet Package

## Installation

1. Clone the repository
2. Install dependencies using npm
3. Check for the installation guide of the zet package documentation.
